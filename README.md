## Build Setup

```bash
# 克隆项目
git clone https://github.com/dianjiu/task-manage-vue.git

# 进入项目目录
cd task-manage-vue

# 安装依赖
yarn install

# 启动服务
yarn run dev
```

浏览器访问 [http://localhost:9528](http://localhost:9528)

## 发布

```bash
# 构建测试环境
yarn run build:stage

# 构建生产环境
yarn run build:prod
```
